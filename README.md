# Bezier Playground

비주얼 노트 캔버스 — Next.js 16 + React 19 + Zustand + Tailwind CSS 4

## 기능

- 무한 캔버스 (줌 30%~200%, 그리드 배경)
- 스크롤로 줌인/줌아웃 (마우스 위치 기준)
- Space + 드래그로 시점 이동
- 좌측 패널에서 노드 추가
- 노드 드래그로 이동
- 노드 삭제 (확인 다이얼로그)

## 실행

```bash
npm run dev
```

## 스택

- Next.js 16 (App Router)
- React 19
- Zustand 5
- Tailwind CSS 4
- shadcn/ui (alert-dialog, button)
- Biome (lint/format)
